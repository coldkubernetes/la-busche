What changed & why

  • Fixed The header (Done / title / ☰) never scrolls away — it's outside the scrolling body, so the hamburger is always reachable, on any page, at any scroll position.
  • Right The drawer slides in from the right, matching the hamburger's position in the header.
  • 5 clear sections Home, Routes and About are plain navigation rows (boxed, with a chevron). Backup is its own boxed nav row leading to a page with two distinct labeled blocks: "Dropbox sync" and "JSON file" — both backup methods, clearly split apart so neither reads as a sub-option of the other. Language is visually different on purpose: no chevron, no navigation — you pick English/Български right there in the menu and it applies immediately.
  • Home added Home (the live tile dashboard outside Setup) is a different destination than Routes (managing saved routes inside Setup), so it gets its own row at the top of the menu — kept alongside the header's Done button, which still works as a fast one-tap exit.
  • Default Setup still opens straight to Routes; Backup and About are one ☰ tap away, and selecting Routes again from the menu returns you there.
  • Try it: tap ☰, then Backup or About, then ☰ again to switch back to Routes. Try tapping Български right in the menu — no page change.
