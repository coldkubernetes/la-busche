What changed & why

  • Flat No hamburger, no drawer. A tab bar sits right under the header, always visible the instant you open Setup — nothing is hidden behind an icon.
  • Taps Home → tap ⚙️ → tap a tab. Same number of taps as the drawer version, but every destination is in view immediately, so it doesn't feel like drilling deeper into the app.
  • Fix Sync and Backup tabs are fixed in the header area, so they're unaffected by how long the route list grows — same problem solved as the drawer version.
  • Default Routes stays the default tab since it's the most-used content; "＋ Add route" stays right above the list, unchanged.
